Geo Standaarden Complex?

Geo standaarden worden buiten (en binnen) het werkveld als complex ervaren...

En dat is misschien ook wel zo...

  • Materie is vrij complex
  • Data providers publiceren de data met een eigen (geo) bril
  • Vertraagde implementatie in gangbare software
  • Interpretatieverschillen
  • Grote bestanden
  • Academische inslag

Projecties

XML gebaseerde standaarden

Hoe gaan we dat oplossen?

  • Data providers bieden de data ook in eenvoudige formaten aan
  • Data portalen bieden functionaliteit om data te extraheren
  • Betere documentatie/voorbeelden standaarden
  • Marktpartijen bieden diensten aan op de open data (arcgis online, nlextract, citysdk)
